=============================================
  WakeelAI - Complete Setup Guide
=============================================

LOCAL (YOUR PC):
-----------------
1. Install Node.js: https://nodejs.org
2. Double-click start.bat
3. Paste your Groq key when asked
4. Open: http://localhost:3000

DEPLOY ONLINE FREE (Railway):
-------------------------------
1. Go to github.com - create account
2. Create new repository called "wakeelai"
3. Upload all these files to GitHub
4. Go to railway.app - sign up with GitHub
5. New Project > Deploy from GitHub > select wakeelai
6. Go to Variables tab, add:
   GROQ_API_KEY = your_groq_key_here
7. Railway gives you a free URL like:
   https://wakeelai-production.railway.app
